Installation

There is no installtion procedure for html .
All we need to do is open your notepad .
IMPORTANT : Make sure to save the notepad file as .html extension to make it a webpage.


Basic Requirements

1. Notepad and/or softwares like Brackets , VSCode , etc. for writing the code .
2. A compatible browser to run the html file . Eg. Chrome, Firefox , Edge , Safari , Netscape ,etc. 3. And a cup of coffee to get started .


Base Design of HTML Page

  • HTML TAG : The first tag to create an html page starts with <HTML> . It is the most important tag used to identify an HTML Document.
    The page starts with <HTML> and ends with </HTML> . So it looks like this :


  • TITLE TAG : The <title> tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab The <title> tag is required in HTML documents!




  • HEAD TAG : The <head> element is a container for metadata (data about data) and is placed between the HTML tag and the BODY tag. Metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information.




  • BODY TAG : The <body> tag defines the document's body. The <body>element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.




Some Attributes of HTML tag's

Attributes provide additional information about elements
Attributes are always specified in the start tag
Attributes usually come in name/value pairs like: name="value"
A list of some html attributes and their uses :
  • <href> : It is an attribute of <a> (anchor) tag . The <href> defines the hyperlink reference of a URL .
    Syntax :
        <a href="URL of the webpage/location of webpage" > Click to goto the webage </a>
  • <src> : It is an attribute of <img> (image) tag . The <src> defines the link/location of the image .
    Syntax :
        <img src="URL of the webpage/location of image" > Click to goto the image </img>
  • <width> & <height> : It is an attribute of <img> (image) tag . The <width> defines the width of the image .
    The <height> defines the height of the image .
    Syntax :
        <img src="abcd.jpg" width=500px height=600px> Click to goto the image </img>
  • <style> : It is can be used in any tag . The <style> defines the style of of elements such as font,image,color,size,etc.
    Syntax :
        <p style="color:red;font:20;" > Here is a random text </p>
